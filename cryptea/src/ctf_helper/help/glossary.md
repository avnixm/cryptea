# Offline Glossary

**Challenge** — A task in your CTF workspace. Each challenge stores metadata, notes, and optional attachments.

**Flag** — The solution token for a challenge. Flags can be stored as plain text or encrypted with a local passphrase.

**.ctfpack** — A portable ZIP archive that contains your challenges, notes, and attachments. Use it for offline backups.

**Snapshot** — An automatic Markdown copy of your notes stored under `~/.local/share/ctf-helper/snapshots/`.

**Tool Module** — A built-in helper like the Hash Digest or Wordlist Generator that runs entirely locally.
